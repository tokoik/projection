# projection
投影テクスチャマッピングの説明用デモプログラム

* GLFW を使っています
* ビルド・実行には OpenCV が必要です
* Web カメラが必要です
* Linux 用 Makefile, Xcode 6 プロジェクト, Visual Studio 2013 ソリューション付き
* マウスの左ボタンドラッグでオブジェクトを回せます
* ホイールで投影像をズーミングできます
* 形状データは Alias OBJ ですがテッセレーションしないのであらかじめ三角形分割してください
